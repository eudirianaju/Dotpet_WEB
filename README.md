# Dotpet_WEB
🐾 Apresento com muito carinho a DOTPET, fruto do meu projeto de conclusão do ensino médio técnico! Imagine um lugar onde cada patinha tem sua história. Meu site não só cataloga adoráveis animais, mas também cria uma ponte para descobertas mais profundas através do nosso app anteriormente visto. Além disso, oferece à ONG um escritório virtual para informações valiosas sobre funcionários, adotantes, adoções, agenda e outros registros. 


Desenvolvido usando HTML, CSS, JS e Java, com um toque de magia em banco de dados SQL via Spring Boot. Junte-se a nós nessa jornada peluda! 🌟🐾
